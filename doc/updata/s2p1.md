## S2P1「迭代前瞻服」更新日志

### 26.6.0-pre2

> [i] 以`^`开头的为管理员可用功能

* 修复
  * 修复云端资源包无法下发正确哈希的问题
  * 修复原版动效增强无法使用的问题
  * 修复附魔「连锁采集」无法使用的问题
  * ^修复村民编辑失效的问题
  * ^修复 CoreProtect 不能记录管理员 FastAsyncWorldEdit 编辑的问题
* 优化与修改
  * 支持最新基岩版入服
  * Java支持版本扩充为1.21~21.1
  * 优化「假人」称号自动穿戴逻辑
  * 提升「回放」系统稳定性
* 移除
  * 移除精英怪系统

### 26.6.0-pre1

* 新增
  * 采用全新网络链路，提升网络稳定性
  * 启用全新 Joyous Menu ，提供更加丰富的菜单界面与功能
  * 启用 MySQL 专业数据库以优化服务器数据存储性能
* 优化修改
  * 更新服务器内核到 Java 1.21.8
  * 更新相关插件
  * 将模拟距离降低到8
* 移除
  * 移除粘液科技及全部附属插件
  * 移除「粘液科技云端资源包」功能
    * 已激活此功能的冒险家将无法继续使用相关功能
