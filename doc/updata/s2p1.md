## S2P1「岭回双壑风」更新日志

### 26.6.0-beta2

* 新增
  * 以下实体现在被禁止进入主城区域——蜘蛛、苦力怕、骷髅、亡灵生物，以及部分来源的箭矢
  * 现在苦力怕在主城的玩家附近爆炸时的爆炸强度自动降为 0
  * 新增 JEI/EMI 的配方查询支持
* 优化与更改
  * 降低进度「极其昂贵！」需求的【强化深板岩】数量
  * 调整主城区域进出提示文本
  * 调整灵魂滞留时长为24小时
  * 优化数据库查询性能
  * 提高允许视距为 12 区块
* 修复
  * 修正进度「怀黍离，希冀禾下凉」的准则文本描述错误
  * 修正「服务器链接」中描述文本错误
  * 修复无法正常召唤假人的问题
  * 修复生物血量显示 UI 过多、影响视觉的问题
  * 修复部分聊天栏链接无法正确显示的情况
  * 修复特定条件下菜单内按钮可被异常取出、点击无效，以及物品可被放入菜单并导致意外丢失的若干交互异常
  * 修复 REI 查询的配方内容错乱
  * 修复称号在部分场景下异常含有 & 字符的问题
  * 修复技能数据异常继承了 S1P2 数据的问题
  * 修复部分情形下出生点异常在海里的问题
  * 修复回放系统无法使用的问题

* 现在支持来自 LittleSkin 的皮肤了
* 完善全新菜单功能
  * 仅限 Java 版，基岩版菜单请等待后续支持
* 正式发布公测版本

### 26.6.0-pre7

* 新增管理权限组「巡查员」
* 优化部分文案
* 优化菜单文案与获取
* 修复称号「假人」不能正常生效的问题

### 26.6.0-pre6

* Joyous 新增邮件发送能力
* 迁移关服邮件至 Joyous 并添加许多新特性

### 26.6.0-pre5

* 修复因为操作系统深度睡眠导致的时间同步问题和网络连接问题
* 修复偶现数据库连接池发生时间跳跃问题
* 修复 StreackLib 变量可见性问题导致的 HTTP 服务不可用的问题

### 26.6.0-pre4

* 兼容 Proxy Protocol 以正确反馈玩家IP
