## S2P1「迭代前瞻服」更新日志

### 26.6.0-beta1

* 新增 LittleSkin 皮肤支持
* 添加菜单的基本功能
* 发布公测版本

### 26.6.0-pre7

* 新增管理权限组「巡查员」
* 优化部分文案
* 优化菜单文案与获取
* 修复称号「假人」不能正常生效的问题

### 26.6.0-pre6

* Joyous 新增邮件发送能力
* 迁移关服邮件至 Joyous 并添加许多新特性

### 26.6.0-pre5

* 修复因为操作系统深度睡眠导致的时间同步问题和网络连接问题
* 修复偶现数据库连接池发生时间跳跃问题
* 修复 StreackLib 变量可见性问题导致的 HTTP 服务不可用的问题

### 26.6.0-pre4

* 兼容 Proxy Protocol 以正确反馈玩家IP
* 修复正版验证相关功能失效的问题
* 扩充基岩版支持为 1.21.130 ~ 26.20
* 暂时下线 Tab 列表中数据展示

### 26.6.0-pre3

* 新增一些进度
* 调整部分文案
* 添加关于海晶石的快速合成配方
* 为「扫地姬」顶部进度条添加可视化倒计时
* 修复进度「终止批处理」的达成条件错误的问题
* 重命名多世界模块的命名空间

### 26.6.0-pre2

> [i] 以`^`开头的为管理员可用功能

* 修复
  * 修复云端资源包无法下发正确哈希的问题
  * 修复原版动效增强无法使用的问题
  * 修复附魔「连锁采集」无法使用的问题
  * ^修复村民编辑失效的问题
  * ^修复 CoreProtect 不能记录管理员 FastAsyncWorldEdit 编辑的问题
* 优化与修改
  * 支持最新基岩版入服
  * Java支持版本扩充为1.21~26.1
  * 优化「假人」称号自动穿戴逻辑
  * 提升「回放」系统稳定性
* 移除
  * 移除精英怪系统

### 26.6.0-pre1

* 新增
  * 采用全新网络链路，提升网络稳定性
  * 启用全新 Joyous Menu ，提供更加丰富的菜单界面与功能
  * 启用 MySQL 专业数据库以优化服务器数据存储性能
* 优化修改
  * 更新服务器内核到 Java 1.21.8
  * 更新相关插件
  * 将模拟距离降低到8
* 移除
  * 移除粘液科技及全部附属插件
  * 移除「粘液科技云端资源包」功能
    * 已激活此功能的冒险家将无法继续使用相关功能
